# Calmi Experiments — Landing Page

Landing page for [Calmi Experiments](https://calmi-experiments.lelekstudio.com) — a self-experimentation app for neurodivergent minds, coaches, and therapists.

## Live URL
[calmi.lelekstudio.com](https://calmi.lelekstudio.com)

## Files
- `index.html` — Main landing page (single file, no build required)
- `robots.txt` — Search engine crawling rules
- `sitemap.xml` — Sitemap for Google indexing
- `vercel.json` — Vercel deployment config and security headers
- `og-image.png` — Social sharing image (1200x630px) — **add this manually**

## Deploy
This site is deployed on Vercel. Every push to `main` triggers automatic redeployment.

## To update
1. Edit `index.html` directly
2. Commit and push to GitHub
3. Vercel deploys automatically in ~30 seconds

## og-image.png
Create a 1200x630px image in Canva with:
- Dark background (#0e0e18)
- Logo: "calmi experiments"
- Tagline: "Small experiments. Real self-knowledge."
- Export as PNG and add to this repo root
